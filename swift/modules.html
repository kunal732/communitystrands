Modules

The SDK ships as a single module. Add one dependency and get everything: all providers, observability, and voice streaming.

StrandsAgents

What it includesExternal dependencies
Agent loop, tools, hooks, multi-agent (Graph, Swarm, A2A), session persistence, MCP aws-sdk-swift, mlx-swift-lm, mlx-swift, mlx-audio-swift, opentelemetry-swift
AWS Bedrock ConverseStream (Claude, Titan, Llama, Mistral)
Anthropic, OpenAI, Google Gemini providers
MLX on-device inference (Apple Silicon)
OpenTelemetry observability, bidirectional voice streaming

Installation

Package.swiftSwift
dependencies: [
    .package(url: "https://github.com/kunal732/strands-agents-swift.git", branch: "master")
]

.target(
    name: "MyApp",
    dependencies: [
        .product(name: "StrandsAgents", package: "strands-agents-swift"),
    ]
)
